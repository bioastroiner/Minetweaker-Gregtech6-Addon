# Commands

---
###### [source code](src/main/java/mods/bio/gttweaker/command/GTCommand.java)

```sh
/mt gt #prints help

/mt gt hand #prints GT data about item
/mt gt hand item #copies an IItemStack refrence unified by gt6 OreDictUnifier
/mt gt hand ore # to get the IOreDict unifier
/mt gt hand material # to get the material coppied

# with no arguments print all existing materials into minetweaker.log
# with argument (either ID or name) it prints verbos info about material into the chat
 /mt gt materials
 /mt gt mats
 /mt gt mat
 /mt gt mt
 
 # similar to materials but with prefixes
 /mt gt prefixes
 /mt gt prefix
 /mt gt op
 
 # lists all of gt6's and its addons' recipemaps
 # with no arguments it lists them all but with argument it gives information about
 # that recipemap (and its IO minimum and maximum)
 /mt gt recipemaps
 /mt gt recipemap
 /mt gt rm
```
